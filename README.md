# Assignment 3

This project compares three algorithms in Java:

Bubble Sort

Merge Sort

Binary Search


The program creates arrays with sizes 10, 100, and 1000.  
It measures execution time using `System.nanoTime()`.

## Algorithms

Bubble Sort is a simple sorting algorithm. It compares nearby elements and swaps them.  
Time complexity: O(n²).

Merge Sort divides the array into parts and merges them back in sorted order.  
Time complexity: O(n log n).

Binary Search works only with sorted arrays. It checks the middle element each time.  
Time complexity: O(log n).

## Result

Merge Sort is faster than Bubble Sort on large arrays.  
Binary Search is very fast because it does not check every element.
